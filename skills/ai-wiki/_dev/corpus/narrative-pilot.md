# 因果推論 — OLS までの問題駆動 narrative

疑いを前提に読む。各ノードは断定的に書くが真理ではない。疑問が湧いたら原典で検証する。forest の中の一本で、他 tree とは peer 関係。

## 記法

```
[?]  問題/障害        [★]  採用された解      [◯]  候補解
[✕]  敗れた候補       [∥]  対立関係         [⛔]  制約/前提
[!]  落とし穴         [∴]  結論/破綻条件    [⤴]  異分野/前段からの借用
[⤵]  副作用          [⟳]  次の問題         [↺]  派生
```

---

## ROOT

```
[?] X の Y への影響を有限データから測りたい
```

以下、この問いから下流に一本の spine が伸びる。各ステップは前のステップの答えから必然的に生じる次の問題であり、独立の章ではない。

---

## 1. まず「Y の予測」理論が要る

```
[⛔] 前提: 確率的モデリング
     データは分布 P からの抽選と見なす
     P / f_X / F_X は同じ情報の 3 表現
│
├─[★] 2 乗ロス下の最良予測 = E[Y]
│     ∥ 対立: 絶対値ロスなら最良予測は中央値
│       ✕ 敗因: 2 乗ロスの数学的扱いやすさ (L² 構造)
│          本質的優位ではなく便宜的選択
│
└─[↺] X の情報を加味したい
      [★] m(X) = E[Y|X] は全関数中の最良予測 (2 乗ロス下)
      性質: 反復期待値 / 関数の外出し / 情報粗視化
      [!] しかし全関数空間での推定は不可能
```

⟳ **だから次の問題**: m(X) の推定困難をどう迂回するか

---

## 2. 迂回戦略 = 線形族への妥協

```
[★] 線形射影 (LP): β₀ = arg min_β E[(Y - X'β)²]
    [★] 主張: LP は CEF の最良線形近似
│
├─[◯] 解の 3 等価表現
│     変分形 / 直交形 E[X(Y-X'β₀)] = 0 / 解析解 β₀ = E[XX']⁻¹ E[XY]
│
├─[⛔] 前提: フルランク条件 (破れ方 2 種)
│     E[XX'] 可逆 ⇔ X に冗長な線形結合が無い
│     破れる典型: ダミー冗長 (定数項 + 既婚ダミー + 未婚ダミー)
│     │
│     ├─[◯] 完全破れ = 完全多重共線性
│     │     係数 β̂ は非一意 (予測値 X'β̂ は一意)
│     │
│     └─[◯] ほぼ破れ = ほぼ多重共線性
│           係数が不安定に爆発
│           ∴ 重大度は目的依存:
│              係数自体が目的 → 深刻
│              予測値のみ目的 → 無害でありうる
│
└─[!] LP ≈ CEF とは限らない
      2 分布が同 CEF でも LP は全く異なる形になりうる
      ∴ 結論の妥当性は分布とリサーチクエスチョン依存
```

⟳ **だから次の問題**: 母集団の β₀ をサンプルからどう推定するか

---

## 3. LP 係数の推定 = OLS

```
[★] OLS 推定量 β̂ = (Σ XX')⁻¹ (Σ XY)
│
├─[!] 深い事実: 独立な 3 動機が同じ推定量に帰着
│     ◯ 目的関数最小化 / ◯ モーメント法 / ◯ プラグイン
│     線形モデルの特殊性、一般の問題では乖離する
│
└─[⤵] 副作用: R² (決定係数) が定義可能になる
      [!] 当てはまりの指標 ≠ 予測精度
      [!] 変数追加で単調増加 (修正 R² でも完全解消せず)
      ∴ 高すぎる R² は変数選択ミスを疑う signal
```

⟳ **だから次に**: 係数の解釈 / 推定量の評価 — 2 方向に分岐

---

## 4. 係数解釈の道具 = FWL 定理

```
[?] 多変数時、特定係数だけ取り出したい

└─[★] FWL 定理 (残差回帰)
      β̂_j = 「他変数で説明されない X_j」 と
             「他変数で説明されない Y」 の回帰
      ⤴ 借用: 線形代数の直交分解
```

⟳ **もう 1 方向の分岐**: β̂ の質を定量評価したい

---

## 5. OLS 推定量の評価   ★ 切り出し候補

```
[?] β̂ はどれぐらい良い推定量か
評価軸 3 (不偏性 / 一致性 / 漸近正規性) + 効率性
│
├─[◯] 不偏性 (有限標本性質)
│     一般: ✕ バイアスを含む
│     例外: CEF が線形 (Y = X'β₀ + e, E[e|X]=0) なら条件付き不偏
│     十分条件: X が全て離散 → CEF は必ず線形結合で書ける
│
├─[◯] 一致性 (漸近)
│     ⤴ ここで初めて必要な前提・道具:
│        ・i.i.d. 前提 (強い仮定、データごとに妥当性検討が必要)
│        ・大数の法則 (LLN): サンプル平均 → 期待値 (確率収束)
│        ・連続関数定理
│     ✓ LLN + 連続関数定理で自動成立
│     要件: E[XX'] フルランク、モーメント有限
│
├─[◯] 漸近正規性
│     ⤴ ここで追加で必要な道具:
│        ・中心極限定理 (CLT): √n スケールで N(0, V) 分布収束
│        ・Slutsky 定理: 確率収束 × 分布収束 の合成
│     ✓ √n(β̂ - β₀) → N(0, V_β)
│     V_β = E[XX']⁻¹ E[XX'e²] E[XX']⁻¹     (一般形)
│     斉一分散下: V_β = σ² E[XX']⁻¹        (簡約形)
│     │
│     └─[↺] 派生: β̂ の漸近信頼区間
│           β̂ ± 1.96 · V̂_β^{1/2} / √n  (95% 漸近 CI)
│           これは「i.i.d. データから任意の統計量を推測する」
│           という一般手法の、OLS β̂ への特化例
│           V_β 未知 → V̂_β で代用
│             (連続関数定理 + Slutsky で正当化)
│
└─[◯] 効率性
      [★] Gauss-Markov 定理: 条件付き不偏推定量の分散下限
      ∴ OLS 評価:
         一般: 下限未達 → 非効率
         例外: 斉一分散 σ²(X) = σ² なら下限達成
      │
      ├─[∥] 対立: GLS (一般化最小二乗)
      │     重み σ⁻²(X) で加重最小化
      │     σ(X) 既知 → Gauss-Markov 下限達成
      │     実行可能 GLS (σ を推定): 不偏性喪失、漸近効率のみ
      │
      └─[↺] 派生: V_β の実際の推定
            [◯] 斉一分散仮定下: σ̂² = Σê²/(n-k)
            [◯] ヘテロスケ対応: HC 系列 (HC0 / HC1 / HC2 / HC3)
                ∴ HC0 < HC1, HC0 < HC2 < HC3
                HC0 は過少評価で非推奨
                HC1-HC3 が実務 default
```
