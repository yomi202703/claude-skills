# Rust

## LENSES

Output schema: `[{file, line, name, kind, lens, safety}]`. clippy single-pass covers L1 + L3 (unreachable_code は default lint).

### L1+L3 — clippy (safety mapped per lint)

```
cargo clippy --message-format=json --all-targets -- -W dead_code -W unused \
  | jq -s '[.[] | select(.reason == "compiler-message") | .message | select(.code.code | test("dead_code|unused|unreachable")) |
      {file: .spans[0].file_name, line: .spans[0].line_start, name: (.spans[0].text[0].text // "?"), kind: .code.code, lens: "clippy",
       safety: (if (.code.code | test("unreachable|unused_import|unused_variable")) then 1 else 3 end)}]'
```

Workspace (`[workspace]` in `Cargo.toml`): same command analyzes all crates.

## TEST_CMD

`cargo test --all-targets`
Build check: `cargo check --all-targets`

## FRAMEWORK_HOOKS (skip — treated as live)

### Language conventions

- `fn main()`
- `#[test]` / `#[bench]` functions
- All functions inside `#[cfg(test)]` modules
- Trait impl methods (`impl Foo for Bar { fn baz() }`) — invoked via trait
- Std trait methods: `Drop::drop` / `Display::fmt` / `Debug::fmt` / `Clone::clone`
- Implementations generated by `#[derive(...)]`

### Explicit allow

- `#[allow(dead_code)]` items — always keep (intentionally unused)
- `#[used]` statics
- `#[no_mangle]` / `extern "C"` (FFI exports)

### Web frameworks

- actix-web: `#[get("/path")]` / `#[post("/path")]` handlers
- axum: handler arg of `Router::new().route("/path", get(handler))`
- rocket: `#[get("/path")]` route macros

### serde

- Fields in `#[derive(Serialize, Deserialize)]` structs
- `#[serde(rename = "...")]` fields

### pub identifiers

- `pub fn` / `pub struct` / `pub enum` in library crates — keep (external API)
- Binary-only crates (`src/main.rs` only): pub items are in scope for detection

### Conditional compilation

- Items behind `#[cfg(feature = "...")]` may appear dead when the feature is disabled — skip

## ENTRYPOINTS

- `src/main.rs`'s `main` function
- `src/lib.rs` pub items (library crate)
- `examples/*.rs` `main`
- `benches/*.rs` `#[bench]`
- `tests/*.rs` `#[test]`
- `[[bin]]` paths in `Cargo.toml`
