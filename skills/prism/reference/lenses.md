# lenses — 目のパレット（prism 独自）

prism が「1つの固定対象を読み直す」ための解釈の角度。手書きの静的成果物。機械は学習しない・自動更新しない。増やす・消す・直すのはあなたが手でここを編集する。

目は角度であってトピックではない。対象を絞り直す（「セキュリティ面だけ見る」）のではなく、対象全体をその角度から読む（「淘汰圧の目で対象を読む」）。同じ対象が目によって違って見えるのが正。

## パレット

- 機構の目 … 対象が「どう動いて成り立っているか」を内部構造・因果で読む。表の主張ではなく動作原理を差す。
- 反証の目 … 対象が壊れる条件・失敗例・前提が崩れる場面を探す。「これが効かなくなるのはどんな時か」。
- 経路依存の目 … 対象が今この形である歴史的経緯を読む。「なぜこうなったか／別の経路もあり得たか」。本質か偶然の固着かを切り分ける。
- 利害・インセンティブの目 … 対象を「誰が得をし誰が損をするか」「誰の動機で推されているか」で読む。技術の顔をした政治を差す。
- 二次効果の目 … 対象そのものではなく、それが下流で可能にする／壊す／不要にするものを読む。「X が成立すると Y が要らなくなる」。
- スケール・限界の目 … 対象を10倍／1/10にしたら何が壊れるか。今の規模に効くものが規模を変えると逆転する点を差す。
- 反実仮想の目 … 「もし対象が存在しなかったら」「代替で置き換えたら」何が変わるかで、対象の正味の効き目を測る。
- 最遠の目 … 対象から最も離れた畑違いの枠（生態系・建築・料理・軍事…）をわざと当てる。アナロジーで普段の枠の外を最大化する振り切り用。
- コストの目 … 対象が隠している代償（運用・認知・移行・技術的負債）を読む。輝いている主張の裏の支払いを差す。
