## Context-cost lessons learned (permanent)

- One 5KB multi-line cell × 3 debug prints = 15KB context
- Multi-row debug dumps (10+ rows) add up fast
- Loop across multiple files + full row prints → 100KB+ context
- **Rule of thumb**: if a debug step touches more than 1 xlsx file OR prints more than 5 rows, redirect to file and grep.
